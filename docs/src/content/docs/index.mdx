This site is the user-facing documentation for the ELLIIIT dataset repository. It is organized in two tracks:

- **Tutorials** for learning how to open, inspect, and interpret the processed RF xarray and the matching per-experiment acoustic `.nc` files.
- **Reference** for the acquisition pipeline, measurement semantics, file layout, and operator-facing workflows.

Start here:

- [Getting Started](tutorials/getting-started/)
- [Use the RF Xarray](tutorials/build-rf-xarray/)
- [Notebook: RF Xarray Structure](tutorials/notebook-xarray-structure/)
- [Notebook: Acoustic Xarray Structure](tutorials/notebook-acoustic-xarray-structure/)
- [Notebook: RF Dataset Overview](tutorials/notebook-overview/)
- [Notebook: Rover Positions](tutorials/notebook-rover-positions/)
- [Notebook: CSI Per Position](tutorials/notebook-csi-per-position/)
- [Notebook: RF And Acoustic At One Position](tutorials/notebook-rf-acoustic-position/)
- [Notebook: CSI Movies](tutorials/notebook-csi-movies/)

The notebook tutorials published here are generated from the source notebooks in `processing/tutorials/` and embedded as read-only HTML exports. The runnable notebook files are also available for download from each tutorial page.
