# Processing

This folder contains the post-acquisition utilities for the dataset repository.

## Layout

- `dataset-download/`: download helpers for published datasets hosted outside the repository
- `parsing/`: scripts that turn raw RF and acoustic captures into processed NetCDF outputs
- `tutorials/`: notebook-based analysis and plotting helpers for the processed datasets

## Typical outputs

- RF dataset: `results/csi_<experiment_id>.nc`
- acoustic dataset: `results/acoustic_<experiment_id>.nc`

Start in the subfolder README that matches your task.
