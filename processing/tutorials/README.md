# Tutorials

This folder contains runnable notebooks and helper utilities for exploring the processed RF and acoustic datasets.

## Contents

- `csi_plot_utils.py`: shared plotting, dataset loading, movie export, and antenna-position helpers used by the notebooks
- `plot_csi_positions.ipynb`: RF dataset overview plots
- `tutorial_xarray_structure.ipynb`: RF xarray structure walkthrough
- `tutorial_acoustic_xarray_structure.ipynb`: acoustic xarray structure walkthrough
- `tutorial_rover_positions.ipynb`: rover position visualizations
- `tutorial_csi_per_position.ipynb`: CSI inspection at individual positions
- `tutorial_rf_acoustic_position.ipynb`: combined RF and acoustic view at one position
- `tutorial_csi_movies.ipynb`: RF movie generation workflow

These notebooks expect processed datasets in `results/`.
